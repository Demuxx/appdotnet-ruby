# Appdotnet

Fooling around with the App.net API.  God help you if you use this thing in its current state.

## Installation

Add this line to your application's Gemfile:

    gem 'appdotnet'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install appdotnet
